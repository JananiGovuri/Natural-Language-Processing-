# NLP Homework 4 – CS5760

This repository contains my solutions for Homework 4 of *Natural Language Processing (Spring 2026)* at University of Central Missouri. The assignment covers both theoretical and practical topics in deep learning for NLP. Feedforward neural networks and perceptrons help in understanding classification and decision boundaries, while XOR with ReLU highlights the importance of non-linearity in solving complex problems.

RNN concepts focus on sequence modeling tasks such as next-word prediction, sentiment analysis, and machine translation, along with challenges like vanishing gradients and solutions using LSTM architectures. LSTM gates (forget, input, output) are important for controlling information flow and preserving long-term dependencies.

Attention mechanisms introduce Query, Key, and Value representations, and scaled dot-product attention improves how models focus on relevant words. Multi-head attention and Transformer encoders further enhance contextual understanding using self-attention and positional encoding.

Through the programming tasks, I implemented a character-level RNN, a mini Transformer encoder, and attention functions, analyzing results using loss curves, generated text, and attention visualizations. This assignment helped me understand how modern NLP models work, why attention is powerful, and how deep learning techniques are applied to real-world language tasks.
